# Smart House Price Prediction System

The Smart House Price Prediction System is a full-stack web application developed using Spring Boot that allows users to estimate property prices based on various input features. The application provides a secure user authentication system and an interactive dashboard for managing and viewing predictions.

## 🚀 Features
- User Registration and Login (Authentication System)
- Secure Login using Spring Security
- Interactive User Dashboard
- Property Price Prediction Module
- RESTful API integration
- Responsive UI using HTML, CSS, and JavaScript


## 🛠️ Technologies Used
- Java (Core Java)
- Spring Boot
- Spring MVC
- Spring Data JPA (Hibernate)
- MySQL (Database)
- HTML, CSS, JavaScript
- Thymeleaf

## 📊 Project Highlights
- Follows layered architecture (Controller, Service, Repository)
- Implements secure authentication and authorization
- Clean and maintainable code structure
- Real-world full-stack development experience

## 🎯 Objective
To build a scalable and user-friendly application that helps users predict house prices efficiently while demonstrating full-stack development skills.
